# CNC Foam Cutter CONTROLLER AND SIMULATOR
This is CNC Foam Cutter for Controlling 2 - 5 Axis Machines and send gcodes via. USB, it can simulate Gcode also, developed by Samupam. Visit www.samupam.com or www.samupam.in to know more about Our Machines.

<img width="1906" height="1024" alt="image" src="https://github.com/user-attachments/assets/1a96970c-a047-4b1f-aaaa-8150733dc7fe" />

1. This Software can control 2,3,4 and 5 Axis CNC Hotwire foam cutter and send gcode over usb. You can simulate gcode before actual cut.
2. There is a option to change The speed and Heat (Our Machines has hardware support for Programmable Heating) during program execution, you can use pause machine then increase/decrease Speed or Head as per your need, it will be changed after execuation of next line of code.
3. The Gui & Graph will adopt as per options choose like 2 axis, 3 Axis, 4 axis or 5 axis. 
4. The Current Controller Supports Two types of firmware Mode 1 and Mode 2, Which covers most of Machines which takes gcodes.
5. To Change Axis Configuration or Mode You must press admin unlock and enter password : admin then only admin settings like change Mode and axis configuration  etc. possible.
6. You can also change axis Names like Y to V etc by pressing following button in Admin mode, You must quit admin mode by pressing button lock configuration then only again all buttons will work,
